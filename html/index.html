  3. 3 按照提示完成安装

Developer Mode

Install manually in developer mode

  1. 1 Download the latest release ZIP file from our GitHub repository
  2. 2 Extract the ZIP file to a folder on your computer
  3. 3 Open Chrome and go to chrome://extensions/
  4. 4 Enable "Developer mode" in the top-right corner
  5. 5 Click "Load unpacked" and select the extracted folder
Download ZIP

开发者模式

通过开发者模式手动安装

  1. 1 从我们的 GitHub 仓库下载最新版本的 ZIP 文件
  2. 2 将 ZIP 文件解压到您电脑上的一个文件夹中
  3. 3 打开 Chrome 浏览器，访问 chrome://extensions/
  4. 4 在右上角启用"开发者模式"
  5. 5 点击"加载已解压的扩展程序"并选择解压后的文件夹
